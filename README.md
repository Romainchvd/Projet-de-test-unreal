Le projet est un parkour en trois niveaux, realises afin de prendre en main le moteur de jeu. Les trois niveaux sont:
- Une discotheque au moment ou la fete bat son plein
- Cette meme discoteque de jour (fermee)
- Cette meme discoteque a l'abandon

Dans chacun des niveaux, le joueur doit collecter toutes les statuettes afin de pouvoir passer au niveau suivant.
/!\ En raison des effets de lumiere, une bonne carte graphique est recommandee pour eviter les lags.

Une fois le jeu termine, le joueur accede a un niveau bonus, qui est le premier niveau debarasse de ses elements de parkour.
